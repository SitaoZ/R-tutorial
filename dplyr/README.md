### mutate
生成新的变量，使用原有的变量进行计算或生成新的变量，在原数据框的基础上返回新的数据框。
可以进行数学运算。

### transmute
添加新的变量，但是返回仅保留新的变量的数据框。
### select
选择变量
### filter
过滤
### summarise
减少变量
### arrange
改变排序
